## 📝 测试报告说明

测试执行完毕后, 测试报告会自动生成到 report 文件夹下

- yaml：测试报告主要用来收集测试结果，用作存储结果数据
- allure：测试报告需启用并正确安装 allure 服务，默认开启
- html：测试报告无需额外配置，默认开启，需手动在文件夹下浏览
- 钉钉 / 飞书 / 邮箱 / ~~-企微-~~：测试报告需正确配置并打开发送开关，测试完成后自动发送

🔍 Yaml Report
-

```yaml
elapsed: 9.003
error: 0
failed: 3
passed: 3
result: Failed
skipped: 3
started_time: '2023-04-01 16:05:37'
total: 9
```
