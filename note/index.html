# ⬇️ 下载

TIP

如果你需要修改代码实现功能定制，我们建议你使用源码

如果你无需功能定制，我们建议你使用 SDK

  • 源码:

    git clone https://github.com/wu-clan/httpfpt.git

  • SDK

    pip install httpfpt

# 🧑‍💻 USE / DEV

  • 如果你正在使用源码，则必须安装相关依赖

    pip install -r requirements.txt

  • 如果你正在使用 SDK，则必须通过 httpfpt 命令创建新项目，命令使用帮助：CLI

    # check out help
httpfpt -h

# create new project
httpfpt --startproject

  • 安装 redis 并启动服务

    Redis Windowsopen in new window

    Linux / macOSopen in new window

    Dockeropen in new window

  • 安装 mysql 数据库（可选，如果你需要本地数据库）
